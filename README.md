# Typescript Template
A minimal TS project template.

## How to Start

Install packages with your preferred package manager.
```
$ npm install
$ pnpm install
$ yarn install
```

If you use **npm** or **yarn**, you will have to update `.husky/pre-commit`.
```
   to your choice of package manager
   v
4  pnpm lint-staged
```
